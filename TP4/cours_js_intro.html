Langage JavaScript — Les bases du langage

La structure de base d’un programme en JavaScript

Lorsque le code JavaScript se trouve dans un fichier séparé (extension .js), la structure générale d’un programme est relativement simple. 

/* zone d'en-tête */

/* déclaration des variables globales */
Declaration1;
Declaration2;

/* définition des fonctions (mot clef function) */

/* CORPS DU PROGRAMME */
Instruction1;
Instruction2;

/* FIN */

Même s'il est possible de déclarer nos variables à n'importe quel moment, il est conseillé de le faire en début de script pour une meilleure lisibilité.

Lorsque le JavaScript est écrit dans une page Web, la structure est différente :

<!DOCTYPE html>
<html>
  <head>
    <title>TODO supply a title</title>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <script>
      // ici les variables globales à la page et les fonctions
    </script>

  </head>
  <body>
    <script>
      // ici les instructions
    </script>
  </body>
</html>

La balise permettant d’ajouter du JavaScript est <script> … </script>. Elle se place souvent dans <head> (variables/fonctions) et dans <body> (instructions).

Les commentaires

Les commentaires permettent la relecture du programme (pour soi-même et pour une équipe). Un commentaire d’en-tête est recommandé.

Deux types :

  • // : commentaire sur la ligne
  • /* … */ : commentaire multi-lignes
/*
 * File:   HelloWorld.js
 * Author: sgibert
 * Created on 14 septembre 2018, 20:19
 * Description : explique ici ce que fait ton programme
 */

Évite les commentaires inutiles qui répètent juste le code.

Déclaration de variables et constantes

JavaScript est faiblement typé : le type n’est pas déclaré explicitement, il est déterminé par l’usage. 

var a = 3;
a = "coucou";

Déclarer :

  • var : variable globale (dans la plupart des cas)
  • let : variable locale dans un bloc { }, globale si hors bloc
  • const : constante (doit être initialisée)
let a;
let b = 4;
let c, d = 3, e;

const f = 3;

var g = "Bonjour";

Types et plage de valeurs

JavaScript a notamment : Number, String, Boolean, Null, Undefined, BigInt, Object, Symbol.

Exemples

let a = 10;
let b = "coucou";
let ok = true;

Attention aux limites de Number (entiers)

let a = 2**53 - 1;
a = a + 20;

Les opérateurs

Opérateurs arithmétiques

  • + addition, - soustraction, * multiplication, / division
  • % modulo, ** puissance

Exemple :

let a = 5, b = 2;
let c = a / b;
let r = a % b;

Opérateurs logiques

  • !, &&, ||
  • ==, !=, <, >, <=, >=

Affectation et incrémentation

  • =, +=, -=, *=, /=, %=, **=
  • ++, --
  • Ternaire condition ? expr1 : expr2
let i = 3;
i++;
i += 1;

let j = 4;
let k = i > j ? 10 : -10;

Fonctions d’entrée/sortie

  • window.alert(message)
  • window.confirm(message) retourne true ou false
  • window.prompt(message, defaut) retourne une chaîne ou null
window.alert("Bonjour");
let ok = window.confirm("Continuer ?");
let saisie = window.prompt("Ton nom ?", "");
