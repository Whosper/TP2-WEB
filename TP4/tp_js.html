TP8 : Les tableaux

Introduction

Dans ce TP, il est demandé d'écrire des fonctions utilitaires permettant de travailler avec les tableaux. Ces fonctions seront ensuite réutilisables dans d'autres TP.

Le tableau utilisé contient N éléments. Il est interdit d'ajouter, d'insérer ou de supprimer une case.

Déclaration du tableau

let N = parseInt(prompt("Combien de case pour le tableau ?"));
let A = new Array(N);
let nbElemRempli = 0;

Aperçu

Le tableau est vide : N cases, toutes vides (valeur undefined).

Organisation du travail

Les fonctions seront codées dans un fichier .js séparé, ajouté au projet.

Pour chaque fonction, respecter les 4 étapes :

  1. Boite noire : réfléchir aux entrées et aux sorties.
  2. Coder la fonction dans le fichier .js.
  3. Ajouter le mode d’emploi de la fonction.
  4. Tester la fonction (inspecteur d’élément).

Exercices

Exercice 1

Écrire une fonction initialiserTab1() qui initialise les N cases d’un tableau avec la valeur 0. À la fin, toutes les cases sont considérées remplies.

Exercice 2

Écrire une fonction initialiserTab2(val) qui initialise les N cases avec une valeur passée en paramètre. À la fin, toutes les cases sont considérées remplies.

Exercice 3

Écrire une fonction remplirTab1(min, max) qui remplit les N cases avec des valeurs aléatoires entre MIN et MAX. Les anciennes valeurs sont remplacées.

Exercice 4

Écrire une fonction remplirTab2(nb, min, max) qui remplit un certain nombre de cases (sur N) avec des valeurs aléatoires entre MIN et MAX, en commençant à partir de nbElemRempli.

Si le nombre de cases à remplir dépasse la capacité du tableau, on remplit jusqu’à la fin (sans ajouter de case).

Exercice 5

Écrire une fonction afficherTab1() qui affiche seulement les cases remplies du tableau.

Exercice 6

Écrire une fonction afficherTab2() qui affiche toutes les cases du tableau (remplies ou non).

Exercice 7

Écrire une fonction afficherTab3(deb, fin) qui affiche les cases du tableau de la position DEB à FIN.

Exercice 8

Écrire une fonction remplirElement(val) qui remplit la première case vide avec une valeur. Si le tableau est plein, l’opération est impossible.

Cette fonction retourne false si l’opération n’a pas pu être effectuée, sinon elle retourne le nombre d’éléments remplis.

Exercice 9

Écrire une fonction saisirTab() permettant à l’utilisateur de saisir un certain nombre d’éléments sur N. Les éléments sont ajoutés à partir de la première case vide.

Cette fonction retourne le nombre de cases remplies.
