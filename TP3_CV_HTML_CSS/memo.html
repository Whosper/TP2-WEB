Règle du mémento

Pour chaque balise repérée, indiquer :

  • Son nom (ex: <strong>)
  • Son rôle
  • Un petit exemple
  • Une trace visuelle (aperçu rendu)

Exemples de balises texte (à adapter selon ce que tu trouves)

<strong>

Rôle : importance forte (mise en valeur sémantique).

Exemple : <strong>Important</strong>

Aperçu : Important

<em>

Rôle : emphase.

Exemple : <em>Accent</em>

Aperçu : Accent

<mark>

Rôle : surlignage.

Exemple : <mark>Surligné</mark>

Aperçu : Surligné

<code> et <pre>

Rôle : affichage de code (inline / bloc).

Exemple : <code>git status</code>

Aperçu : git status

<blockquote> et <q>

Rôle : citations longue / courte.

Citation longue.

Une citation courte dans une phrase.

<abbr> et <time>

Rôle : abréviation + date/heure.
